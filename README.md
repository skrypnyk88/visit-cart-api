1) git clone git@github.com:skrypnyk88/visit-cart-api.git
2) cd visit-cart-api
3) bundle install
4) rails db:migrate (Elasticsearch повинен бути запущений локально)
5) rails db:seed (Elasticsearch повинен бути запущений локально)
6) rails s
